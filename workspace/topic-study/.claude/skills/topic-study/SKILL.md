# News Causality Skill

ニュースのトピックを構造的に理解するための因果連鎖分析Skill。Kenshi(PE/M&A consulting業務)の思考補助ツールとして設計。

## 起動条件

ユーザーが以下のような発話をしたら、このSkillを起動する:

- ニュースの一文やトピック名を提示し、解釈を求める
- 「○○の影響で△△」という因果関係の妥当性を問う
- 「なぜ○○が起きているのか」と背景を求める
- 関連株への影響を尋ねる

## 全体フロー(全自動)

トピックを受け取ったら、以下を**1ターンで一気通貫**に実行する。各セクションを途中で省略しない。

```
[Step 0] 一言でいうと (200字程度の散文サマリー)
   ↓
[Step 1] 因果連鎖の仮説構築 (自然な日本語ノード + 接続詞)
   ↓
[Step 2] 自己レビュー (順接/逆説の論理整合性チェック)
   ↓
[Step 3] 必須媒体ファクトの収集と整理 (日経・日経ビジネス・東洋経済を必ず横断 + 1次情報1件以上)
   ↓
[Step 4] 論説の分岐抽出 (コンセンサス vs 異論)
   ↓
[Step 5] 株式インパクト (業績変動メカニズム + 早見表)
   ↓
[Step 6] Gmail送付 (kenkenkenshi54@gmail.com → kamei.kenshi@adlittle.com)
```

## ボリューム指針

このSkillの出力は**読み手が一読で理解できること**を最優先する。密度よりも明晰さ。

- **専門略語に頼らない**(Brent → 原油価格 / JKM → アジアLNGスポット価格 / P → 売値 / Q → 販売量 / ボラ → 価格変動 / JGM → JERAのトレーディング部門 など)
- 各ノードの文字数は拘らない。**自然な日本語**で読めることを優先
- ただし、各ノードは**1つの事象・1つの命題**に絞る(複文を詰め込まない)
- 業界カテゴリは最大4、各カテゴリ代表銘柄は2〜3まで
- Step 0(一言でいうと)で読み手が構造を掴めるかが最重要

## Step 0: 一言でいうと (新設・最重要)

因果連鎖を出す**前**に、トピックの核を200字程度の散文で示す。これが読み手の地図になる。

### 書き方

- **構造の核を一文で**: 「Aは○○側、Bは△△側、Cは□□側。だから〜」のような対比軸・分類軸を明示
- **平易な日本語**: 専門用語・略語は使わず、新聞記事のリード文のような自然な文章で
- **2-3行・200字程度**: 短すぎるとサマリーにならず、長すぎると因果連鎖と重複する
- **「一言でいうと」「核心は」「要するに」のような明示的な書き出し**で始める

### Step 0の例

#### 例1(イラン戦争×INPEX/JERA/関電)

> **一言でいうと**: 戦争で原油・LNG価格が上がる局面で、INPEXは「自分で掘って売る側」だから売値が上がって直接儲かる。JERAは「世界最大のLNGバイヤー」として、長期契約で押さえた玉を高値で動かすトレーディングで儲ける。一方、関西電力は「使うだけ」で、原発比率が高くLNG感応度も低いので、ダメージもアップサイドも小さい中立。3社は「販売側・仲介側・需要側」と立ち位置が違う。

#### 例2(円安×自動車×半導体)

> **一言でいうと**: 円安で恩恵を受けるのは「ドル建て売上が大きく、生産は国内」の企業。トヨタは国内生産比率がまだ3割超あり、円安1円で営業利益が400億円動く構造。逆に、海外生産比率が高い日産・ホンダは恩恵が薄く、輸入部材コスト増で逆風になる場面もある。半導体製造装置は売上の8割超が海外なので、円安は素直に追い風。

### Step 0の注意

- ❌ 結論をぼかす(「複雑な要因がある」「一概に言えない」)
- ❌ 因果連鎖の冒頭をそのままコピペ
- ❌ 専門用語の羅列
- ✓ **対比軸を明示**(○○ vs △△、上流 vs 下流、輸出 vs 輸入)
- ✓ **数値を1つ入れる**(感応度・依存度・シェアなど、構造を象徴する数字)

## Step 1: 因果連鎖の仮説構築

### ノード(事象)の書き方

- **自然な日本語で書く**。文字数の上限は設けない。ただし複文・接続詞を含めて1ノードにしない
- **専門略語に逃げない**。Brent → 原油価格、ボラ → 価格変動、P/Q → 売値/販売量、と平易に展開
- **主語と述語を明確に**。何が・どうなった、を1つの事象として書く
- **時制は現在/完了の自然な日本語**で統一

### 良いノードと悪いノード

| ❌ 略語に頼った例 | ✓ 平易に展開した例 |
|---|---|
| Brent 60→112ドルに急騰 | 北海ブレント原油が1バレル60ドルから112ドル超に急騰 |
| JKMが$24.8/MMBtuに倍増 | アジアのLNGスポット価格が3月に倍増し、100万BTUあたり24.8ドルに到達 |
| 価格ボラ拡大が機会の源泉 | 価格の振れ幅が大きいほど、長期契約とスポットの差で稼ぐ機会が増える |
| JGM経由でトレーディング機能 | シンガポール拠点のJERA Global Markets(約300人)がLNGの売買仲介を担う |

### 番号と階層

- ①②③...と丸数字で番号を振る
- 各ノードの後に接続詞をインデントして記載

```
①[事象を自然な日本語で]
  └〔順接〕**その結果**
②[次の事象]
  └〔理由付け〕**なぜなら**
③[背景の事実]
```

### ノード数の目安

| トピックの複雑さ | ノード数 |
|---|---|
| 単純(2段階の因果) | 4〜6 |
| 中程度(複数経路) | 8〜12 |
| 複雑(多段階・多変数) | 12〜18 |

20を超える場合は、サブグループに分解して階層構造で示す。

### 階層化のパターン

トピックが複雑な場合、フェーズや主体ごとに分けて構造化する:

```
**【マクロ価格の経路】**
①〜⑤

**【A社側の経路】**
⑥〜⑩

**【B社側の経路】**
⑪〜⑮

**【総括】**
⑯〜⑰
```

### 反証/弱点の付記

各ノードまたは末尾セクションで、以下の観点から反証・弱点を明示する。

**反証パターン:**

1. **過去事例の否定**: 「過去の◯◯戦争では実際には〜だった」
2. **代替経路の存在**: 「中東依存度は実は低下している」「代替原料がある」
3. **緩衝材の存在**: 「在庫がある」「長期契約で保護されている」
4. **規模感の問題**: 「影響額は売上の◯%にすぎない」
5. **タイムラグ**: 「即時影響ではなく、数四半期遅れる」

**書き方例:**

```
**論理の弱点(DD的視点での突っ込みどころ)**

- **③**: ホルムズ海峡の封鎖は、過去の中東紛争では限定的・短期的に終わるケースが多い
- **⑥**: 日本のナフサ調達は中東比率が以前より低下、韓国・東南アジアからの輸入も多い
- **⑫**: 「停止」は強すぎる表現、実態は「価格転嫁+納期遅延」が一般的な反応
```

### 接続詞の使い分け(6分類)

#### 1. 順接 (A→Bが自然な流れ)

| 接続詞 | ニュアンス |
|---|---|
| その結果 | 直接的な因果 |
| ゆえに | 論理的帰結を強調 |
| よって | やや硬い、結論寄り |
| したがって | フォーマル、論理連鎖 |
| そこで | 行動主体が対応した |
| すると | 時系列の継起 |

選定基準:
- 物理的・機械的な因果 → その結果、すると
- 論理的帰結 → ゆえに、したがって
- 主体的な対応行動 → そこで、よって

#### 2. 逆説 (A→Bが想定と逆)

| 接続詞 | ニュアンス |
|---|---|
| しかし | 一般的な逆接 |
| ところが | 予想外の展開 |
| にもかかわらず | 強い逆接 |
| とはいえ | 部分的な留保 |
| 一方 | 対比の提示 |

**重要ルール:** 逆説を使ったら、**直後または直前のノードで「なぜ逆方向に動くのか」の理由を必ず付ける**。

#### 3. 理由付け (背景説明)

| 接続詞 | ニュアンス |
|---|---|
| なぜなら | 標準的な理由提示 |
| というのも | やや柔らかい |
| というのは | 説明的 |
| そもそも | 前提に遡る |
| 背景には | 構造的説明 |

#### 4. 補足 (並行・追加)

| 接続詞 | ニュアンス |
|---|---|
| しかも | 強調を伴う追加 |
| さらに | 程度の追加 |
| 加えて | フォーマルな追加 |
| ここで | 補足説明の導入 |
| そして | 中立的な追加 |
| そのうえ | 累積的追加 |

#### 5. 並行 (別経路での同時進行)

| 接続詞 | ニュアンス |
|---|---|
| 他方 | 対比的な並行 |
| 一方で | 中立的な並行 |
| 同時に | 時間的並行 |
| 並行して | 機能的並行 |

#### 6. 結論 (収束)

| 接続詞 | ニュアンス |
|---|---|
| 結果として | 標準的 |
| 結局 | やや断定的 |
| つまり | 言い換え/要約 |
| 最終的に | 時系列の終点 |

## Step 2: 自己レビュー

以下のチェックを内部的に実行し、不備があればStep 1に戻って修正する:

1. **Step 0と因果連鎖の整合**: 一言サマリーで述べた構造軸が、因果連鎖でも一貫して維持されているか
2. **順接で繋がっているか?** A→Bが順接なら、「AならばBは自然」と言えるか
3. **逆説の場合、理由付けが成立しているか?** 「AにもかかわらずB」の場合、なぜ逆方向に動くのか説明があるか
4. **跳躍がないか?** 中間の因果が省略されて結論に飛んでいないか
5. **専門略語に逃げていないか?** Brent/JKM/P/Q/ボラ等の略語が残っていれば、平易な日本語に展開する
6. **世間一般の認識と乖離していないか?** 専門的すぎる/独自すぎる解釈になっていないか
7. **「だから」「なので」のような口語的接続詞を使っていないか?**
8. **同じ接続詞ばかり連続していないか(単調になる)?**

## Step 3: 必須媒体ファクトの収集と整理

### このスキルの肝: 「日経・日経ビジネス・東洋経済」の3媒体を必ず横断する

PE/M&A視点の分析では、**主要経済メディアの記事をファクトとして引く**ことが信頼性の根幹になる。検索回数を惜しまず、以下の3媒体を必ず1回以上ヒットさせる。

| 媒体 | 強み | 検索クエリ例 |
|---|---|---|
| 日本経済新聞 (nikkei.com) | 一般経済・産業の網羅性、業界動向の定点観測 | `site:nikkei.com [トピック]` |
| 日経ビジネス (business.nikkei.com) | 業界深掘り、経営者インタビュー、構造的論点 | `site:business.nikkei.com [トピック]` |
| 東洋経済オンライン (toyokeizai.net) | 企業財務分析、批判的論調、ランキング | `site:toyokeizai.net [トピック]` |

**有料記事の扱い**: 冒頭部分しか読めなくても、見出し・リード文だけで情報価値は大きい。論調の傾向を読む。

### ファクトの整理方法 (出力フォーマット)

検索結果を**論点別**に整理する。「ノード番号 → 該当記事の要約 → 出典」の形で表形式にまとめる。

```markdown
### 必須媒体ファクト

#### 日本経済新聞のファクト
| ノード | 記事内容(要約) | 出典 |
|---|---|---|
| ⑤ | Brent原油は4月7日に112.95ドル、4月23日に105.07ドルで推移 | [記事タイトル](URL) (日経, 2026/4/X) |
| ⑭ | JERAは日本のLNG輸入の約40%を扱う世界最大のバイヤー | [記事タイトル](URL) (日経, 2026/3/X) |

#### 日経ビジネスのファクト
| ノード | 記事内容(要約) | 出典 |
|---|---|---|
| ⑫ | JERAは中東リスク回避のため、米国LNGに9兆円投資する戦略を打ち出している | [記事タイトル](URL) (日経ビジネス, 2026/3/X) |

#### 東洋経済オンラインのファクト
| ノード | 記事内容(要約) | 出典 |
|---|---|---|
| ⑪ | INPEXのイクシスLNGは利益の7-8割を占め、オセアニア中心で中東依存は低い | [記事タイトル](URL) (東洋経済, 2026/X/X) |

#### 1次情報 (IR・官公庁・業界統計から最低1件)
- 関西電力の2024年度電源構成: 原子力25.8%、石炭21.6%、LNG等17.1% — [統合報告書2025](URL) (1次情報)
```

### 補完: 階層3・階層4の媒体

3媒体で見つからない場合や、論点を補強する場合に使う:

- **テレビ東京BIZ**: WBS・モーサテ等の現場取材・アナリスト視点
- **ロイター日本版**: 国際的視点、市場反応の即時性
- **Bloomberg日本版**: 機関投資家視点、グローバルマクロ
- 業界専門紙(日刊工業新聞、化学工業日報等)、アナリストレポート公開部分

**SNS・個人ブログは原則使わない**。

### 1次情報の必須要件

3媒体ファクトとは別に、**最低1件は1次情報を含める**。階層2以下のメディアだけで構成された分析は、PE/M&A視点での精度を担保できない。

具体的には:
- 当事者企業のIR資料(決算短信、説明会資料、適時開示)
- 官公庁の統計・公表資料(経産省、金融庁、財務省、日銀等)
- 業界団体の統計・白書
- 国際機関のレポート(IEA、OECD、IMF等)

検索しても1次情報が見つからない場合は、その旨を明示し「**メディア情報のみによる仮説**」であることを断る。

### 検索の3段階フロー

#### Stage 1: 全体俯瞰 (1〜3検索)

```
検索: [トピック] 影響
検索: [トピック] 最新 [現在の年]
検索: [トピック] 業界
```

#### Stage 2: クリティカルノード補強 + 3媒体横断 (各媒体最低1検索)

クリティカルノードの判定基準:
1. 数値・固有名詞を含む(定量的裏取りが必要)
2. 反証されると全体が崩れる(根幹前提)
3. 逆説の前後(理由付けの裏取りが特に重要)
4. 株式インパクトの根拠
5. ユーザーが特に強調しているノード

各媒体について、クリティカルノードに対する検索を**少なくとも1回ずつ**実行する。

```
site:nikkei.com [ノードのキーワード]
site:business.nikkei.com [企業名] [事象]
site:toyokeizai.net [業界] [トピック]
```

#### Stage 3: 反論探索 (3〜5検索) — 必須

```
検索: [トピック] 影響なし
検索: [トピック] 限定的
検索: [トピック] 過剰反応
検索: [トピック] 楽観
検索: [トピック] 一過性
```

**Stage 3を必須化する理由:** 論説の分岐がない=コンセンサスだけ拾うになる。意図的に反論を探さないと両論併記は成立しない。

### 検索回数の目安

| トピック規模 | 総検索回数 |
|---|---|
| 単純 | 8〜12回 (3媒体×2 + 反論3 + 1次情報1) |
| 中程度 | 12〜18回 |
| 複雑 | 18〜25回 |

### 引用とCitation

- **直接引用は15語未満**、1ソースから1引用まで(著作権配慮)
- **媒体名・日付・記事タイトル**を明示
- **数値・固有名詞は正確に転記**
- 大半は**自分の言葉で要約**(言い換え)する

### 補強できなかったノードの扱い

検索しても裏付けが取れなかったクリティカルノードは、**正直に「検証できず」と明示**する。仮説のまま強弁しない。

## Step 4: 論説の分岐抽出

ニュースを読んだ結果、見解が分かれる論点があれば明示する:

```markdown
**【論説A: コンセンサス見解】**
- 主張: ...
- 根拠: (出典○○)

**【論説B: 異論】**
- 主張: ...
- 根拠: (出典△△)

**【評価】**
- 短期(0-3M): A優勢/B優勢
- 中期(3-12M): 〜
- 長期(1Y+): 〜
```

論説は**最大3つまで**(A/B/Cで足りる)。冗長になるなら統合する。

## Step 5: 株式インパクト

### 基本原則

#### ニュースの因果と株の影響を分離する

Step 1〜4で構築した因果連鎖と、Step 5の株式インパクトは**論理的に別レイヤー**で扱う。

- 因果連鎖: 「世界で何が起きるか」のメカニズム
- 株式インパクト: 「企業業績がどう変動し、株価にどう波及するか」

#### 業績変動メカニズムは因果連鎖と同じフォーマット

「自然な日本語ノード+接続詞」で売値・販売量・コストの変動を表現する。「○○関連銘柄」「××セクターへの追い風」のような曖昧表現は禁止。

### 出力フォーマット

#### 業界カテゴリの上限: 3〜4まで

## お手本: イラン戦争とシンナー供給

### トピック: イラン戦争の影響でシンナーの供給が止まる、という説の検証

> 業績変動メカニズムの分析であり、投資推奨ではない。

#### 一言でいうと

イラン戦争でホルムズ海峡が緊迫すると、中東経由の原油・ナフサが滞り、ナフサを原料とするトルエン・キシレン(=シンナーの主成分)の供給が逼迫する、というのが世間で語られるロジック。ただし日本のナフサ調達は中東依存が以前より低下しており、国内BTX設備の内製分も大きい。**「停止」より「価格転嫁+納期遅延」が現実的な姿**で、塗料・印刷インキ業界には短期的な利幅圧迫として効いてくる。

#### 因果連鎖

**【主たる伝播経路】**

①イラン戦争が勃発する。
  └〔順接〕**その結果**
②ホルムズ海峡の航行が緊迫化する。
  └〔理由付け〕**なぜなら**
③中東原油の主要航路だから。
  └〔順接〕**ゆえに**
④原油・ナフサのタンカー航行が停滞する。
  └〔順接〕**したがって**
⑤日本へのナフサ輸入が逼迫する。
  └〔理由付け〕**というのも**
⑥日本は依然として中東への原油依存度が9割超だから。
  └〔順接〕**よって**
⑦ナフサ価格が高騰する。
  └〔補足〕**ここで**
⑧ナフサは芳香族(BTX)の原料である。
  └〔順接〕**ゆえに**
⑨トルエン・キシレンの供給が逼迫する。
  └〔補足〕**そして**
⑩トルエン・キシレンはシンナーの主成分。
  └〔逆説〕**しかし**
⑪塗装・印刷需要は底堅く、需要は減らない。
  └〔結論〕**結果として**
⑫シンナーの市中供給が逼迫する。

①'戦争報道で関係者の不安心理が拡大する。
