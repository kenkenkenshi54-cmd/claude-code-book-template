# オムロン リレー事業 用途別競争力分析（車載／産機／配電盤・再エネ）

**作成日**: 2026-05-02
**ディール文脈**: Carlyle 譲渡（事業価値810億円、2026/10/1 完了予定、新社名 Aratas Corporation）後の対象事業価値レンジ評価のための、競争力の構造分解

---

## 0. 分析フレームと結論先出し

### 0-1. 採用フレーム

ユーザー指示の論理連鎖を **フォワード（用途→収益）** と **リバース（収益→用途）** の双方向で検証する。

```
[フォワード] 用途 → システムKBF → スイッチ技術選択 → リレーKBF → オムロン充足度 → シェア／収益性
[リバース ] 観察される低収益／低シェア → どのリレーKBF未充足が原因 → そのKBFはどのシステムKBFから生じている → その用途のどの特性が決定づけたか
```

両方向で矛盾なく説明できれば「KBFは充足してるのに収益性が低い」という単純見せかけの矛盾も解消できる。

### 0-2. 結論先出し（後の論証で裏付け）

1. **用途間で勝敗が劇的に分岐する**：オムロンは PLC I/Oリレー（MY/G2R）／SSR（G3PH）／PhotoMOS（G3VM）／安全リレー（G9SX）で勝ち、車載HV-DCコンタクタ／AC電磁接触器／中国EV・BESS／北米PV-BESS で負ける。
2. **「KBF充足度」の概念を分解すべき**：性能KBF（電圧／電流／寿命／規格）はオムロンも他社も大半で充足する。差がつくのは（i）**コスト充足度**、（ii）**先行認定実績の充足度**、（iii）**地理・チャネル近接性の充足度**、（iv）**ロードマップ追従性（800V/1500V級など次世代KBF）の充足度**。これら「広義KBF」の差で勝敗が決まっている。
3. **対 TE/宏发/Sensata の質的違い**：性能KBFでは互角だが、TE は「60年の航空宇宙系譜＋北米デファクト」で**先行認定KBF**が圧倒的、宏发は「中国EV/BESS生態系の中での専業30億個規模」で**コスト＋地理的近接性KBF**が圧倒的、Sensata Gigavac は「Kilovac DNAスピンアウト＋ニッチ特化」で**双方向・特殊用途KBF**が圧倒的。オムロンは「FAブランド＋センシング系譜」で**自社チャネル抱合せKBF**が圧倒的だが、それが効かない領域では総合点で負ける。
4. **収益性低位の構造的原因**：H4（規模劣後）× H3（参入タイミング遅延 10-15年）× H2（R&D系譜の重電不在）の三重ロックイン。性能KBFを満たしてもシェアが取れず、シェアが取れないので規模が出ず、コストKBFも満たせない、という負のループ。
5. **逆方向検証の整合**：DMB全体 OPM 0.3%（FY2024）の観察は、「勝てる領域（高OPM）が小さく、負ける領域（低OPM）でボリュームを稼いでいる」というミックス劣化説で完全に整合する。性能技術力が低いから赤字なのではない。

## 1. 用途別の論理連鎖（フォワード）

### 1-1. 車載リレー

#### A. 用途の網羅整理

EVインバーターの主スイッチは IGBT/SiC であり「リレー」ではない。リレーが入るのは**インバーター周辺の電力経路の入切**である。

|---|------|---------|-------------|-------------|

| (b) | HV主回路 | プリチャージリレー（突入電流抑制） | 400-1500VDC / 10-50A | 小型メカDCリレー |
| (c) | 急速充電 | CCS/NACS/GB/Tインレットコンタクタ | 800-1000VDC / 350-500A | メカ式HV-DCコンタクタ |

### 2-3. 質的違いの言語化

#### vs **TE Connectivity（Kilovac）**
- **TEの強み = "60年の航空宇宙＋軍事＋宇宙の重電系譜"**。1964年Carpinteria創業のKilovacが、NASA・潜水艦・防衛用途で蓄積したアーク消弧物理を、2008年Tesla Roadster量産時にそのまま EV用 LEV200 として転用→事実上の北米HV-DC標準を確立。
- **オムロンとの最大の質差 = (iii)先行認定KBF と (vii)標準策定権KBF**。オムロンは性能KBF(i)では追いついても、「Tesla初代から積み重ねた安全設計トラックレコード」「TEが業界用語として "EV200" "LEV200" を語らせるだけのブランド」を持っていない。
- **オムロンの相対優位 = (v)チャネルKBF**。TEは大電流コネクタ統合で攻めるが、FA抱合せでの I/Oリレー pull は持たない。

#### vs **宏发 Hongfa**
- **宏发の強み = "中国30億個/年、専業60年級、厦漳生態系集積"**。2024年売上141億元（≈3,000億円、オムロンDMB全体の約3倍）、毛利率36.2%、純利率11.6%。中国EV/BESS生産集積（BYD/CATL/NIO/Sungrow）に対する地理的近接性＋専業スケール＋コスト圧倒で勝負を決めている。
- **オムロンとの最大の質差 = (ii)コストKBF と (vi)地理近接性KBF**。性能KBFでは互角でも、ユニットコストで構造的に勝てない。中国EVが「自国部品優先」のサプライチェーン政策をとる中、宏发を出し抜く外資は宿命的に困難。
- **「KBFを充足しているのに勝てない」という構図はここで最も顕著**。オムロンの G9EK は性能仕様で宏发HFE と互角でも、(ii)+(iii)+(vi) の3層で構造的に劣る。

#### vs **Sensata Gigavac**
- **Sensataの強み = "Kilovac DNAスピンアウト＋ニッチ特化＋双方向の先駆者"**。2002年GIGAVAC LLC は **「Kilovacを構築した経営チームが起こしたスピンアウト」**。GTM400/500 は双方向セラミック封止HV-DCコンタクタの事実上の用語定義者。2018年 Sensata $233M買収、買収前5年 CAGR 30%超、年商 $80M。
- **オムロンとの最大の質差 = (i)性能KBFの最先端 と (iii)先行認定KBF**。BESS 双方向／V2G のような「次世代KBF」に最初から特化していたため、北米IPP・データセンター・特殊BESSでデフォルト指名される。
- **オムロンの相対優位 = (v)チャネルKBF（FA／産業）と (i)性能（汎用領域）**。ただしGigavacが攻める領域はそもそもオムロンの主戦場ではない。

#### vs **Panasonic Industry**
- **パナの強み = "重電・松下電工系譜のカプセル接点＋水素ガス＋日米OEM深耕"**。EV-A（2010年代前半）、EV-B（2000A遮断）でトヨタ・ホンダ・テスラ北米・欧州OEMを押さえる。

- **逆にPanasonicも(v)FAチャネルKBFは弱い**。MY/G2R系の国内デファクトはオムロンが保持。

## 3. 用途間で結果が異なる構造的原因（ディープ）

7仮説を独立評価した結果、**H4×H3×H2 の三重ロックイン**が最強の説明セット。

### 3-1. H4：規模・コスト構造（最強）

| 比較 | データ |
|------|-------|
| 宏发 | 売上 141億元（≈3,000億円）、年産 30億個、160シリーズ、40,000規格、毛利 36.2%、純利 11.6% |
| Schneider IA | EBITA 14.8%、年商 €10B規模 |
| Panasonic Industry | OPM 5.2%（リレー単体はもっと低い） |
| **オムロン DMB** | 売上1,054億円、OPM 0.3%（FY2024） |

宏发のリレー専業売上だけでオムロン DMB全体（リレー含むスイッチ／コネクター／センサ等）の3倍。コモディティ領域で勝てる規模ではない。

### 3-2. H3：参入タイミング遅延（10-15年）

| 製品 | 投入時期 |
|------|---------|
| Kilovac EV200 | 2000s半ば〜（Tesla Roadster 2008量産で採用） |
| Panasonic EV-A | 2010年代前半（Tesla電池契約 2010/1） |
| 宏发 HFE系HVDC | 2012-15（中国EV立上げと同時にスケール） |
| **オムロン G9EK** | **2025年1月本格広報** |

Tier 1 PPAP承認サイクル 2-3年＋設計凍結後のSOPロックインで先行優位が固定化される業界特性下、10-15年遅れは構造的に挽回不能。

### 3-3. H2：R&D系譜の重電不在

| 各社の出自 |
|-----------|
| オムロン（1933、立石電機）：誘導型保護継電器→マイクロスイッチ→センシング＆コントロール。**重電・大電流アーク消弧の蓄積は薄い** |
| 宏发（1984、厦門）：**リレー専業40年級**。Power/Auto/HVDC/Latching/Signal で全カテゴリ世界首位 |
| TE Kilovac（1964、Carpinteria CA）：**航空宇宙・宇宙・潜水艦・軍事用60年級** → 2007年Tyco分割→TE |
| Sensata Gigavac（2002）：**Kilovacを構築した経営チームがスピンアウト**、Kilovac DNA正統後継 |
| Panasonic（松下電工系譜）：重電・配線器具・接点技術。**水素ガス封入セラミック封止カプセル** |

オムロンには「ガス封入アーク消弧物理」のR&D系譜が存在しない。一方で、SSR（半導体）・PhotoMOS（光半導体）・安全ロジック（制御アルゴリズム）はオムロン本流の隣接領域。**勝ち負けの分岐が R&D系譜の隣接性で説明できる**。

### 3-4. なぜ三重ロックインが「同時に」効くのか

時系列の因果連鎖：

```
H3 先行参入
  → 量産経験曲線取得
  → H4 規模効果でユニットコスト低下
  → 専業 R&D を継続強化（H2）
    → さらに性能で先行
      → OEM 認定取得（先行認定 KBF）
        → 設計凍結後のロックイン（参入障壁化）
          → 後発の H3 リカバリ困難
            → 規模出ない → コスト負け
              → 結果：性能 KBF 充足でも勝てない
```

オムロンが「勝つ領域」（PhotoMOS／SSR／PLC I/F）はこのロックインを**回避できた領域**：(a) 自社FAチャネルが pull を生む、(b) 出自の隣接性がある、(c) 標準策定権を持っていた（MY/G2R）。

オムロンが「負ける領域」（HV-DC／AC接触器／中国EV/BESS）は**完全にロックインの内側**：(a) 外部チャネル必要、(b) 出自が遠い、(c) 標準策定権を持たない、(d) 参入が遅い。

## 4. リバース検証：観察される収益性 → 原因にきれいに繋がるか

ユーザー指示の双方向検証。フォワードの結論が、観察事実から逆算しても矛盾なく辿れるかを確認。

### 4-1. 観察される定量事実（出発点）

| 指標 | 値 | 出典 |
|------|----|----|
| DMB 売上 FY2024 | 1,054億円 | オムロン決算短信 |
| DMB 営業利益 FY2024 | 3億円（前期比 -90.3%） | 同上 |
| DMB 営業利益率 FY2024 | 0.3% | 同上 |
| DMB ピーク売上（FY2020） | 1,389億円 → FY2024 1,054億円（5年で -24%） | 同上 |
| カーブアウト価格 | 事業価値810億円（PSR 0.77） | 日経 2026/3/30 |

### 4-2. リバース連鎖

#### Q1. なぜ OPM 0.3% に落ちたか？
A1. 売上ミックスの劣化＋原材料高＋構造改革コスト。

#### Q2. ミックスはなぜ劣化したか？
A2. 高OPM領域（PhotoMOS／SSR：推定 OPM 25-35%）の絶対規模が小さい一方、低OPM領域（汎用PCBリレー／ボディ系／補機系：OPM 5%以下）でボリュームを稼ぐ構造が、低OPM領域の単価下落でさらに薄利化。

#### Q3. 低OPM領域はなぜ単価下落したか？
A3. 宏发等中国系の規模＋コストでの価格攻勢（H4）と、12V系MOSFET置換による需要縮小（用途特性）。

#### Q4. なぜ高OPM領域の規模を拡大できなかったか？
A4. PhotoMOS／SSRはニッチ寡占で市場規模自体が小さい（PhotoMOS 世界 4-7億ドル）。市場規模が大きい HV-DC（成長領域）への移行が遅れた（H3：参入タイミング 10-15年遅延）。

#### Q5. なぜ HV-DC への参入が遅れたか？
A5. (a) R&D系譜が重電不在で参入準備に時間（H2）、(b) Tesla/BYD/VW のEVプラットフォーム立上げ期に既に Panasonic／TE／宏发 がOEM認定済み、(c) 自社FAチャネルが効かない領域での新規顧客開拓に時間。

#### Q6. なぜ自社FAチャネルが効かないか？
A6. 車載は OEM/Tier1、BESS/PV は エネルギー事業者・PCSメーカーが指定主体。FAエンジニアの指名権が及ばない。一方で、PLC I/F や SSR では自社IAB（Sysmac PLC国内2位）の抱合せが pull を生む。

#### Q7. なぜ MY/G2R/PhotoMOS でだけ標準策定権を持てたか？
A7. オムロン創業期（1933年〜）の国内リレー黎明期に主導していた領域だから（H5）。一方、IEC TC23（低圧開閉）、CHAdeMO（DC急速充電）、ASIL/ISO 26262（自動車機能安全）の発起メンバーには入っていない。

→ **リバース連鎖が用途特性まで滑らかに辿れる**。論理矛盾なし。

### 4-3. 矛盾チェック

| 一見矛盾しそうな点 | 解消 |
|-------------------|------|
| 「KBF充足してるのに収益性低い」 | 性能KBFは充足。広義KBF（コスト／先行認定／ロードマップ／チャネル／地理／標準策定）の一部または全部が未充足。広義 KBF の一部欠如で勝負がついている |
| 「世界Top5圏外なのに、PhotoMOS世界Top2」 | 用途別では正反対。HV-DC は Top5圏外、PhotoMOS は Top2。同じ「リレー」でも領域別で勝敗が分岐 |
| 「祖業なのに OPM 0.3%」 | 祖業＝重要だが、市場構造が変わって（中国EV化、半導体置換、宏发規模）祖業の戦い方では勝てなくなった。R&D 系譜の慣性が逆に重荷 |
| 「PSR 0.77 と低マルチプル」 | 構造劣後を織り込んだ妥当評価。バリューアップの主戦場は (i) PhotoMOS／SSR の高OPM領域でのROIC改善、(ii) 中国コスト構造取り込み、(iii) HV-DC ロードマップ追従。Carlyle PE型ガバナンスでこれが達成できれば 1.5-2 倍の EVマルチプル拡大シナリオ |

## 5. 統合的結論

### 5-1. 競争力の構造

オムロンのリレー事業の競争力は、**「自社FAチャネル × センシング隣接系譜 × 国内デファクト先行」**の3要素を持つ領域（PLC I/F、SSR、PhotoMOS、安全リレー）で **強く、世界でも通用する**。

逆にこれら3要素を持たない領域（HV-DC コンタクタ、AC電磁接触器、中国EV/BESS、北米PV-BESS）では、**TE Connectivity（航空宇宙系譜＋北米デファクト）**、**宏发（中国専業30億個規模）**、**Sensata Gigavac（Kilovac DNA＋ニッチ特化）**、**Panasonic（重電系譜＋日米OEM）**という、それぞれ「**独自のロックイン構造を持つ**」競合に対して構造的に劣後する。

### 5-2. 「KBF充足度」を再定義した上での評価

性能KBF（電圧／電流／規格）は概ね互角〜部分劣後。**勝敗を分けるのは広義KBF**：(ii)コスト、(iii)先行認定、(iv)ロードマップ、(v)チャネル、(vi)地理、(vii)標準策定権。

- **TE 比**：(iii)先行認定と(vii)標準策定権で大差。HV-DCで Top5入りは構造的に困難。
- **宏发 比**：(ii)コストと(vi)地理近接性で大差。中国EV/BESS市場の主戦場では勝負にならない。
- **Sensata 比**：(i)双方向／V2Gのような最先端ニッチKBFで遅れ。北米IPPデフォルト枠は取れない。
- **Panasonic 比**：(iii)日米OEMの先行認定で劣後。日系市場でも主力供給社になれない。
- **逆にオムロンが他社に勝つ領域**：(v)FAチャネルと(i)PhotoMOS/SSRの隣接系譜による性能。

### 5-3. 収益性の論理整合

DMB OPM 0.3% は、上記の領域別勝敗パターンの結果として整合的に説明できる：

```
高OPMで勝つ領域（PhotoMOS／SSR） ：規模が小さく利益貢献限定
低OPMで負ける領域（汎用PCB／HV-DC／コンタクタ）：規模はあるが薄利／中国攻勢で価格下落
　　　　　　　　　　　　　　　　　　　　　　　　　　　　＝
セグメント全体 OPM が 5%級 → 0.3%へ崩壊（FY2022→FY2024）
```

**「性能KBFは充足しているのに収益性が低い」は表面的矛盾**で、広義KBF分解で完全に解消される。

### 5-4. Carlyle カーブアウトの戦略合理性

本分析と整合する戦略仮説：
1. **Carlyle 傘下**で、(a) 中国生産・コスト構造取り込み（H4対策）、(b) HV-DC ロードマップ加速（H3部分挽回）、(c) PhotoMOS／SSR の高OPM領域での更なる集中投資による Bolt-on 機会（H1強化）。
2. オムロン本体は **IAB（制御機器）＋HCB＋SSB＋DSB** に集中、ROIC 12% 目標を阻害していたDMBから解放。
3. ただし **H3先行認定の挽回は構造的に困難**で、Carlyle 傘下 5年でも HV-DC で世界Top5入りは現実的ではない。**より現実的なバリューアップは PhotoMOS／SSR／産業 SSR の高OPM 領域での集中＋ニッチ M&A**。

### 5-5. 追加 DD で詰めるべき論点

1. **DMB 内のリレー単独売上・OPM**（リレー／スイッチ／コネクター／センサの製品分類別分解）：公開開示なし。Carlyleの取得価格根拠としても最重要論点
2. **PhotoMOS／SSR の単独売上と OPM**：高OPM領域がDMB全体のいくらの利益貢献をしているかが、バリューアップ戦略の中核
3. **HV-DCリレー（G9EK等）のシェア・受注バックログ・主要顧客**：自動車OEM／Tier1（VW、トヨタ、Tesla、BYD、CATL等）との供給契約条件、長期契約の有無
4. **本体オムロンとの内部取引比率（IAB ↔ DMB）**：(v)チャネルKBFのスタンドアローン化後の継続条件
5. **中国深圳工場の依存度**：生産能力の何%、品目別の北米／日本工場との代替可能性、関税回避のためのSCM再構築余地
6. **Carlyleバリューアッププラン（公開非開示）**：Bolt-on買収候補（EM Devices、富士通コンポーネント、Song Chuan、American Zettler等）の検討状況、Capex計画
